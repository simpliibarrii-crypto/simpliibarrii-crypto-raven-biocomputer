# Roadmap

## v0.1, bounded workstation

- Deterministic sequence-analysis tools
- Per-run workspace and artifact hashes
- JSpace policy gate
- Raven ecosystem receipts
- CLI, FastAPI, MCP, Docker, and Hugging Face demo surfaces

## v0.2, reproducible notebook cell

- Signed tool manifests
- Resource quotas and timeouts
- Dataset mounts with read-only provenance
- Notebook execution through a separately isolated worker
- Home for AI desktop status panel

## v0.3, verified biology computer

- Pluggable Firecracker, gVisor, Kata, or E2B execution backend
- Visual desktop streaming for approved scientific applications
- App-specific state verifiers and machine-checkable task completion
- Signed run bundles and tamper-evident event chaining

## v1.0, open biology computer

- Genomics, transcriptomics, proteomics, structure, microscopy, and literature workstations
- Shared benchmark suite for biology computer-use agents
- Institutional policy packs
- Reproducible scientific run export
- Human approval console for wet-lab and clinical boundaries
